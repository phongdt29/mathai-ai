📝 Ghi chú & cảnh báo

⚠️ Đây là ước tính dựa trên cấu trúc prompt và giới hạn token trong code, chưa đo thực tế. Hiện OPENAI_API_KEY đang trống nên hệ thống chạy fallback tất định (0 token). Để có số chính xác: nạp API key thật rồi đọc bảng ai-log (lưu tokensUsed mỗi lần gọi).
  • Chi phí lớn nhất nằm ở lần tạo giáo trình (sinh toàn bộ bài tập một lượt — output có thể chạm trần 16K token của gpt-4o-mini).
  • Co giãn tuyến tính theo số module / buổi / bài tập — tăng cấu hình thì output và chi phí tăng theo.
  • Đổi model: gpt-4o đắt hơn ~17–25×; gpt-5 output cũng ~17× so với mini.
  • Phát sinh ngoài luồng: AI Solver (giải bài) và Chat Tutor mỗi lượt là 1 call riêng (~600 token/lượt), không nằm trong luồng "hoàn thành buổi học" cơ bản.
  • Chấm điểm: câu trắc nghiệm chấm tất định (0 token); chỉ câu tự luận / trả lời ngắn mới gọi AI.
